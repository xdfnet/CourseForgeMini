# CourseForge优课工坊 - AI智能课程生成器

CourseForge优课工坊是一个基于AI的智能课程生成工具,可以快速创建结构化的课程内容。

## 主要功能

- 生成课程大纲
- 生成详细的课程内容
- 自动保存生成的内容到本地文件

## 技术栈

- Python 3.x
- PyQt6 用于GUI界面
- OpenAI API (使用Moonshot模型)用于内容生成

## 使用方法

1. 运行程序后,输入以下信息:
   - 课程标题
   - 目标用户
   - 章数 (1-6)
   - 节数 (1-6)

2. 点击"生成课程纲要"按钮生成课程大纲。

3. 点击"生成课程内容"按钮生成详细的课程内容。

4. 生成的内容将保存在`D:\temp\[课程标题]`目录下。

## 注意事项

- 确保已安装所有必要的Python库。
- 需要有效的OpenAI API密钥。
- 生成过程可能需要一些时间,请耐心等待。
