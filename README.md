# 优课工坊（CourseForge）

优课工坊是一款革命性的课程创作与学习平台,旨在为教育者和学习者提供一个创新、高效的环境。现在,我们推出了基于AI的智能课程生成工具,进一步简化课程创作过程。

## 项目概述

优课工坊集成了:

- 强大的课程创作工具,包括AI智能课程生成器
- 丰富的学习资源库
- 智能的个性化推荐系统
- 互动式学习社区

无论您是想创建引人入胜的课程,还是寻找适合自己的学习内容,优课工坊都能满足您的需求。

## 核心功能

1. **AI课程锻造**: 
   - 快速生成课程大纲
   - 自动创建详细的课程内容
   - 支持多媒体内容整合
2. **资源探索**: 海量优质课程资源,覆盖各个学科和领域。
3. **智能推荐**: 基于AI算法,为用户推荐最适合的学习内容。
4. **社区互动**: 促进学习者与教育者之间的交流与合作。

## 技术栈

- Python 3.x
- PyQt6 用于GUI界面
- OpenAI API (使用Moonshot模型)用于内容生成

## 使用方法

1. 运行程序后,输入课程标题、目标用户、章节数量等信息。
2. 点击"生成课程纲要"按钮生成课程大纲。
3. 点击"生成课程内容"按钮生成详细的课程内容。
4. 生成的内容将自动保存在指定目录下。
